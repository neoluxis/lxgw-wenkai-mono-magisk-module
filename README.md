# LXGW Wenkai on Android Phones

在安卓设备上设置系统字体为 落霞孤鹜文楷字体

### Disclaimer

需要先安装 KernelSU/SukiSu 等管理器，然后下载源码，运行 `packer.sh` 即可打包。也可以从release下载打包好的文件。

若不打包 `system/product`，则只会修改软件的字体，而不会修改 Google 软件的字体；本仓库中替换了 Google 的字体。
